# 快速开始

## 同步代码 

```
http://git.300.cn/gouuse/web/scrm.git
```

## 安装依赖

```js
npm install --registry=http://192.168.5.216:4873
```

## 运行开发环境

> 对应后端开发环境接口
```js
npm run dev
```

> 对应后端测试环境接口
```
    npm run dev:test
```

> 对应后端预发布环境接口
```
    npm run dev:pre
```

> 对应后端正式环境接口
```
    npm run dev:pro
```
