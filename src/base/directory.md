# 通用项目结构


```
/
├── build/                  配置
├── config/                 配置
├── static/                 资源目录，第三方及无需编译的资源
├── src/
│   ├── assets/             资源目录，字体、图标、css、less等
│   ├── components/         公用组件，针对各模块之间或全局可复用的组件
│   ├── global/             全局配置，公共方法、请求、主菜单等配置
│   ├── i18n/               多语言，部分项目未使用
│   ├── pages/              模块页面，包含各个模块独立路由配置及模块相关组件
│   ├── router/             主路由配置
│   └── store/              vuex相关，部分项目未使用
├── App.vue                 入口
└── main.js                 入口
```
